## MCQ 1
Question: What is recursion in programming?
A) A function that calls another function
B) A function that calls itself
C) A loop that repeats indefinitely
D) A conditional statement that checks for a condition
Correct Answer: B) A function that calls itself

## MCQ 2
Question: What is the base case in a recursive function?
A) The case where the function calls itself
B) The case where the function terminates
C) The case where the function starts
D) The case where the function repeats
Correct Answer: B) The case where the function terminates

## MCQ 3
Question: What is the main advantage of using recursion?
A) It reduces the time complexity of a program
B) It increases the space complexity of a program
C) It breaks down a complex problem into smaller sub-problems
D) It makes the program more difficult to understand
Correct Answer: C) It breaks down a complex problem into smaller sub-problems

## MCQ 4
Question: What is a queue in data structures?
A) A linear data structure where elements are inserted and deleted from the same end
B) A linear data structure where elements are inserted from one end and deleted from the other end
C) A non-linear data structure where elements are inserted and deleted from any end
D) A non-linear data structure where elements are inserted from one end and deleted from the other end
Correct Answer: B) A linear data structure where elements are inserted from one end and deleted from the other end

## MCQ 5
Question: What is the difference between a queue and a priority queue?
A) A queue follows the FIFO principle, while a priority queue follows the LIFO principle
B) A queue follows the LIFO principle, while a priority queue follows the FIFO principle
C) A queue does not have any priority, while a priority queue has priority
D) A queue has priority, while a priority queue does not have any priority
Correct Answer: C) A queue does not have any priority, while a priority queue has priority

## MCQ 6
Question: What is the purpose of the enqueue operation in a queue?
A) To delete an element from the queue
B) To insert an element into the queue
C) To check if the queue is empty
D) To check if the queue is full
Correct Answer: B) To insert an element into the queue

## MCQ 7
Question: What is the purpose of the dequeue operation in a queue?
A) To insert an element into the queue
B) To delete an element from the queue
C) To check if the queue is empty
D) To check if the queue is full
Correct Answer: B) To delete an element from the queue

## MCQ 8
Question: What is a circular queue?
A) A queue where the last element points to the first element
B) A queue where the first element points to the last element
C) A queue where the elements are arranged in a linear fashion
D) A queue where the elements are arranged in a non-linear fashion
Correct Answer: A) A queue where the last element points to the first element

## MCQ 9
Question: What is a double-ended queue?
A) A queue where elements can be inserted and deleted from one end
B) A queue where elements can be inserted from one end and deleted from the other end
C) A queue where elements can be inserted and deleted from both ends
D) A queue where elements cannot be inserted or deleted
Correct Answer: C) A queue where elements can be inserted and deleted from both ends

## MCQ 10
Question: What is the main advantage of using a double-ended queue?
A) It reduces the time complexity of a program
B) It increases the space complexity of a program
C) It allows for efficient insertion and deletion of elements from both ends
D) It makes the program more difficult to understand
Correct Answer: C) It allows for efficient insertion and deletion of elements from both ends
